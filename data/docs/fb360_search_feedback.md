# How to search and filter feedback in FB360

## Overview
FB360 accumulates a large volume of feedback over time. The search and filter capabilities allow you to find specific feedback records quickly, identify patterns across clients, and surface relevant insights for product planning or client conversations.

## Step-by-step instructions

### Step 1 — Access the feedback list
From the FB360 dashboard, click **All Feedback** in the left navigation to see the full feedback list. By default, entries are sorted by submission date with the most recent first.

### Step 2 — Use the search bar
Type keywords into the search bar at the top of the feedback list. The search covers:
- Feedback description text
- Feature titles linked to the feedback
- Client names
- Feedback IDs

### Step 3 — Apply filters
Use the filter panel on the left to narrow results:

**By feedback type:**
- Feature Request
- Bug Report
- Satisfaction
- NPS

**By sentiment:**
- Positive
- Neutral
- Negative

**By client:**
- Select one or more clients from the dropdown

**By date range:**
- Use the date picker to set a from and to date

**By feature:**
- Filter feedback linked to a specific feature request

**By revenue opportunity:**
- Set a minimum revenue opportunity threshold (e.g. show only feedback with opportunity > $100,000)

### Step 4 — Use the Copilot for intelligent search
For more nuanced searches, use the **Copilot** bar above the feedback list:
- "Show me negative feedback from Banking clients in the last quarter"
- "What features are most requested by Platinum tier clients?"
- "Find feedback related to API performance from clients in the APAC region"

The Copilot interprets natural language and applies the appropriate filters automatically.

### Step 5 — Save a search
If you run the same search regularly, click **Save search** to bookmark the filter combination. Saved searches appear in the left navigation under **My searches**.

### Step 6 — Export results
To export feedback results to a spreadsheet, click **Export** and choose CSV or Excel format. Exports include all visible columns based on your current filter state.

## Common search scenarios

### Finding all feedback for a specific client
- Click the client name filter and select the client
- Sort by date descending to see the most recent feedback first

### Finding high-value feature requests
- Set feedback type to Feature Request
- Set minimum revenue opportunity to $500,000
- Sort by revenue opportunity descending

### Preparing for a QBR
- Filter by client
- Set date range to the last 12 months
- Export to CSV for inclusion in the QBR report

### Identifying negative sentiment trends
- Set sentiment to Negative
- Set date range to the last 3 months
- Group by category to see which areas are generating the most negative feedback

## Tips
- Combine multiple filters for precise results — each filter narrows the result set further.
- The date range filter uses submission date, not the date the feedback was discussed with the client.
- Saved searches are personal — they are not shared with other users unless exported.
